ShipCompliant - Ruby API Client

The ShipCompliant gem is a API Client for the SOAP API provided by ShipCompliant. This gem has been created for fast implementation and supports the 9 main API calls.

Guides | RDoc | CoreService V1.2

Installation

Add this line to your application's Gemfile:

gem 'ship_compliant'

And then execute:

$ bundle install

Add this to config/initializers/ship_compliant.rb

ShipCompliant.configure do |c|
  c.partner_key = ENV.fetch('SHIP_COMPLIANT_KEY')
  c.username = ENV.fetch('SHIP_COMPLIANT_USER')
  c.password = ENV.fetch('SHIP_COMPLIANT_PASS')
end
